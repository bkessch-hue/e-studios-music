# E-Studios

Music production studio website featuring videos, tracks, and music lessons.

## Features

- **Music Videos** — Watch studio sessions and performances
- **Tracks** — Stream and explore the latest music
- **Music Lessons** — Register for guitar, piano, drums, vocals, bass, and production lessons
- **Responsive Design** — Works on desktop, tablet, and mobile

## Tech Stack

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox)
- Vanilla JavaScript
- SVG graphics

## Setup

Simply open `index.html` in a browser or serve with any static file server.

```bash
# Using Python
python3 -m http.server 8000

# Using Node.js
npx serve .
```
